Project: Android CI/CD & Cloud Deployment Pipeline

To validate and demonstrate the complete DevOps workflow, I initially used a simple HTML application instead of an Android application. This allowed me to focus on implementing the end-to-end CI/CD pipeline, including:

GitHub for source code management
Jenkins for CI/CD automation
Docker for containerization
Docker Hub for image storage
Kubernetes for container orchestration
AWS EC2 for cloud deployment
Prometheus & Grafana for monitoring

Once the pipeline is fully validated, the same workflow can be extended to build and deploy a real Android application (APK/AAB) with only minor changes to the build stage.
